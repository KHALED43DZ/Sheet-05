# حساب المبيعات - نادي الأعوان

تطبيق PWA لحساب مبيعات نادي الأعوان.

## الملفات

```
├── index.html     ← التطبيق الرئيسي
├── manifest.json  ← إعدادات PWA
├── sw.js          ← Service Worker (يعمل بدون إنترنت)
├── icon-192.png   ← أيقونة 192×192
└── icon-512.png   ← أيقونة 512×512
```

## رفع على GitHub Pages

1. أنشئ مستودع جديد على GitHub (مثلاً: `sales-app`)
2. ارفع جميع الملفات
3. اذهب إلى **Settings → Pages**
4. اختر **Branch: main** ثم اضغط **Save**
5. انتظر دقيقة وسيكون الرابط:
   ```
   https://اسم-المستخدم.github.io/sales-app/
   ```

## تثبيت التطبيق

- **Android**: افتح الرابط في Chrome → سيظهر زر "تثبيت" في الأعلى
- **iPhone**: افتح في Safari → زر المشاركة → "إضافة إلى الشاشة الرئيسية"
